Tags HTML

table

A tag <table> define uma tabela em HTML.

  • <tr> define cada linha da tabela;
  • <th> define o cabeçalho da tabela;
  • <td> define cada célula da tabela.

Exemplo:

Saída:

Animal Cor Idade
cachorro branco 2
gato preto 3

del

A tag <del> define que algum texto foi deletado do documento. Normalmente os navegadores colocam um risco no meio da palavra ou frase

Exemplo:

Saída:

Eu gosto do inverno!

ins

A tag <ins> define que algum texto foi inserido no documento. Normalmente os navegadores representam isso colocando underline abaixo do texto

Exemplo:

Saída:

Eu gosto de calor!

sub

A tag <sub> define um subtexto, que é um texto menor que aparece abaixo do nível normal dos textos comuns.

Exemplo:

Saída:

Este texto contém um subtexto.

sup

A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé.

Exemplo:

Saída:

Este texto contém um texto sobretexto.

blockquote

A tag <blockquote> especifica uma sessão que é citada de outra fonte.

Exemplo:

Saída:

Citação do site do WWF:

Há 50 anos, o WWF protege o futuro da natureza. Organização de conservação líder mundial, o WWF atua em 100 países e é apoiado por 1,2 milhão de membros nos Estados Unidos e cerca de 5 milhões no mundo todo.

q

A tag <q> define uma citação curta. Normalmente o navegador coloca "aspas" por volta da citação.

Exemplo:

Saída:

O objetivo do WWF é: Construir um futuro onde as pessoas vivam em harmonia com a natureza. Esperamos que tenham sucesso.

abbr

A tag <abbr> define uma abreviação ou um acrónimo, por exemplo: HTML, CSS, OMS, etc.

Exemplo:

Saída:

A OMS foi fundada em 1948.

address

A tag <address> define o contato de informação do autor de um documento ou artigo.

Exemplo:

Saída:

By Mariele Longato

cite

A tag <cite> define um título de alguma obra como por exemplo uma música, um poema, um livro, uma pintura, etc.

Exemplo:

Saída:

O Grito de Edward Munch. Pintado em 1893.

bdo

BDO significa Bi-Directional Override, serve para mudar a direção do texto.

Exemplo:

Saída:

Este parágrafo irá da esquerda para a direita

Este parágrafo irá da direita para a esquerda

code

A tag <code> define um pedaço de código de computador. O navegador usa a fonte monospace padrão.

Exemplo:

Saída:

A tag HTML button define um botão clicável.

pre

A tag <pre> define texto pré-formatado. Preserva os espaços e quebras de linha.

Exemplo:

Saída:

                    Texto em um elemento pré
                    é exibido    preservando
                     os espaços e
                    quebras de linha

kbd

A tag <kbd> define inputs de teclado, normalmente mostrado com fonte monospace.

Exemplo:

Saída:

Pressione Ctrl + C para copiar o texto (Windows).
