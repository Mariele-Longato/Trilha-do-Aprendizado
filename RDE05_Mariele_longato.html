Hyperlinks

Hyperlink ou link

É uma referência dentro de um documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico.

Os links podem ser remotos, internos, locais, com endereço de correio ou arquivos.

Marcação de Hiperlinks

A marcação de hiperlinks acontece utilizando a tag <a> e a propriedade href que faz referência para o conteúdo desejado.

Sintaxe:

Hiperlinks externos ou remotos

São links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL (Uniform Resource Locator) para a referência que desejamos.

Sintaxe:

Hiperlinks internos

São links que apontam a um lugar diferente dentro da mesma página e possibilta acessar diferentes parágrafos ou seções quando o documento é extenso.

Sintaxe:

Hiperlinks locais

São links que apontam para outras páginas dentro de um mesmo site.

Sintaxe:

Hyperlinks em imagens

É possível realizar hyperlinks imagens envolvendo a tag de imagem com a tag de hiperlink.

Sintaxe:

Hyperlinks para downloads de arquivos

São links que apontam para arquivos e possibilitam realizar download destes arquivos.

Sintaxe:

Hiperlynks para endereço eletrônico

São links que apontam para um endereço eletrônico.
