Branches e Hospedagem

Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • novas funcionalidades;
  • correção de problemas;
  • alteração de código.

Branch Master ou Main

  • braço principal de um projeto em um repositório

Branch Secundária

  • braço secundário de um projeto em um repositório.

Ao criar um repositório no GitHub, automaticamente ele é inicializado com a branch main, que é a parte principal do projeto. As demais branchs que forem criadas posteriormente são as secundárias, e podem existir várias em um único projeto.

Quando um projeto está sendo desenvolvido por várias pessoas, o ideal é que cada uma desenvolva seus códigos em uma branch e ao final realiza-se a junção com a branch main, por meio de um processo denominado merge, que utiliza o pull request.
